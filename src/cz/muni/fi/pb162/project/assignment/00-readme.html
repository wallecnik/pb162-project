Povinné úkoly:

  1. Vytvořte třídu Demo spustitelnou z příkazového řádku, která bude v balíku cz.muni.fi.pb162.project.demo. Program vypíše na standardní výstup text "Hello World!". Ověřte, že jde třída skutečně spustit z příkazového řádku.
  2. Zdokumentujte třídu Demo pomocí JavaDoc.
  3. Otestujte třídu Demo pomocí testovací třídy ProjectTest. Po úspěšném otestování odevzdejte do odevzdávárny a nechte zkontrolovat cvičícího.
